# Track Specification: UI Localization (pt-BR)

## Goal
Translate all user-facing text in the application to Portuguese (Brazil).

## Requirements
1.  **Scope**: All Pages and Components.
    -   Layout (Navigation, Header)
    -   Login Page
    -   Dashboard
    -   Inventory Management
    -   Batch Management
2.  **Language**: Portuguese (pt-BR).
3.  **Approach**: Direct replacement of hardcoded strings (Hard-coded i18n).

## Key Terminology
-   Dashboard -> Painel
-   Inventory -> Inventário
-   Batches -> Lotes
-   Settings -> Configurações
-   Catalog Item -> Item do Catálogo
-   Lot Number -> Número do Lote
-   Expiry Date -> Validade
-   Quantity -> Quantidade
-   Location -> Localização
