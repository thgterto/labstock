# LabControl 2.0 Web

## Product Overview
LabControl 2.0 Web is a modern, offline-first inventory management system for laboratories, featuring comprehensive tracking and management tools.

## Technical Architecture
- **Framework**: React 19
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM v7 (HashRouter)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Data Persistence**: LocalStorage (Service layer abstraction)

## Key Features
- Dashboard with analytics
- Inventory management (Catalog, Items)
- Offline-first capability
